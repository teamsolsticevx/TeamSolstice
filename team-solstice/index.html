How the recruitment score works
Every game mode returned by The Hive's stats API is scored independently, then summed for the overall score. Within a mode: wins are weighted heaviest, kills add a smaller amount, and win rate contributes a bonus scaled by matches played. A modest activity term rewards a larger sample size without letting raw playtime dominate skill. Field names are matched defensively (wins, kills, deaths, played and common variants) since the public API doesn't fix an exact schema per mode.
